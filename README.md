# AEVE-REC-Session REST Server

Endpoint | HTTP Method | Description
-------- | ----------- | -----------
`/run_rec/<name>` | `PUT` | Run video. `<name>` is name of volunteer
`/stop_rec` | `GET` | Stop video.
